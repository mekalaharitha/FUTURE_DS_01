# Business Sales Performance Analytics Dashboard

## Objective
Analyze business sales data to identify revenue trends, top-selling products, category performance, and regional sales performance.

## Tools Used
- Microsoft Excel
- Pivot Tables
- Pivot Charts

## Dataset
Global Superstore Sales Dataset

## Dashboard Components
1. Sales by Region
2. Sales by Category
3. Top 10 Products by Sales
4. Sales Trend (2015-2018)

## Key Insights
- West region generated the highest sales.
- South region generated the lowest sales.
- Technology is the best-performing category.
- Canon imageCLASS 2200 Advanced Copier is the top-selling product.
- Sales showed significant growth from 2016 to 2018.
- 2018 recorded the highest annual sales.
